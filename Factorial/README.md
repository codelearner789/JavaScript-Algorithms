# Objective
    Write a function that takes in a number and returns the factorial.

# Example Details
    5! = 5 * 4 * 3 * 2 * 1 = 120

# Example Returns
* yourFunction(5) returns 120.
* yourFunction(4) returns 4.
* yourFunction(0) returns 1.
* yourFunction(10) will return 3628800.

# Hints
* 1 parameter
* for loop
